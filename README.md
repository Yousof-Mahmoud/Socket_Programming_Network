# Socket_Programming_Network
a simple server client socket programming project for clinical reservatiom
# Prerequests
## 1- Python V. 3.8.8  <br/>
## 2- pip V. 21.3.1 <br/>
## 3- For Client-Server Socket establishment:
```
 >> pip install socket
 >> pip install sys
 >> pip install select
 >> pip install errno
``` 
## 4- For DB connection
```
>> pip install PyQt5
>> pip install mysql-connector 
```
